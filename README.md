Reywen is a library for building bots on the Revolt.chat platform, written in Rust. It provides simple abstraction libraries for easily implementing features, while allowing for more direct interaction with the Revolt APi and Websocket.

## Status
This is the last major version of ReywenV2, this code will stop receiving support and updates effective 25/06/23. There are many issues with the current code base that required an entire rewrite, beyond version 0.1.10 will be entirely new API methods and names and so old code **will not work with older versions**.
